Configuração da Integração com Google Sheets

🔗 URL do Apps Script (Web App)

Cole aqui a URL gerada ao publicar o Apps Script como "Web App". Ela será salva no navegador e usada para carregar e salvar dados no Google Sheets.

ℹ️
Como configurar passo a passo 1. Crie uma planilha no Google Sheets.
2. Acesse Extensões → Apps Script e cole o código do arquivo AppScript.gs.
3. Clique em Implantar → Nova implantação → Web App.
4. Escolha Executar como: Eu e Quem pode acessar: Qualquer pessoa.
5. Copie a URL gerada e cole no campo acima.
6. Na primeira vez, autorize as permissões solicitadas pelo Google.
7. Clique em Salvar URL e depois em Testar conexão.
🔒
Segurança e CORS O Apps Script precisa retornar o cabeçalho CORS correto para funcionar com o GitHub Pages. O código fornecido já inclui esse tratamento. Em caso de erro de CORS, certifique-se de ter reimplantado o script após qualquer alteração de código.
